# アパレル系ブランドの業務改善ツール開発

## 期間
2022年7月〜2024年3月（1年9ヶ月）

## プロジェクト概要
アパレル企業向けの商品企画管理ツールの新規開発に従事。商品情報や企画進行を管理する業務システムで、要件定義から設計・実装・リリース・運用まで一貫して担当。

単独開発体制において、仕様に基づいた設計・実装を中心に、開発全体を担う形でプロダクトを構築。

---

## 役割・担当領域
- 要件定義に基づき、設計・実装・テスト・リリースまでを単独で担当
- フロントエンド・バックエンド・インフラを横断したフルスタック開発
- 開発プロセス改善およびCI/CDの導入

---

## 実績・取り組み

### ① 単独での開発完結
- 開発メンバー1名体制で、設計・実装・テスト・リリースまでを一貫して担当
- 要件をもとにシステム設計へ落とし込み、開発を推進

---

### ② CI/CDの導入による開発効率・品質向上
- GitHub Actions を用いたCI/CDパイプラインを構築
- テスト・ビルド・デプロイの自動化により、リリース作業の効率化とヒューマンエラーの削減を実現

---

### ③ フロントエンド・バックエンドの設計実装
- Vue.js / Nuxt.js（TypeScript）によるフロントエンド開発
- Django REST Framework を用いたバックエンドAPIの設計・実装
- 業務要件に基づいたデータ構造設計およびAPI設計を実施

---

### ④ クラウドインフラの設計・構築
- AWS（ECS / RDS / DynamoDB）を用いたインフラ構築
- Dockerを利用したコンテナ化および開発・本番環境の整備

---

## 技術スタック
- フロントエンド: Vue.js / Nuxt.js / TypeScript
- バックエンド: Python / Django REST Framework / pytest
- インフラ: AWS（ECS / RDS / DynamoDB）
- CI/CD: GitHub Actions
- その他: Docker

---

## 担当工程
要件定義、基本設計、詳細設計、実装、単体テスト、結合テスト、運用・保守
