# エネルギー系企業のDXツール開発

## 期間
2021年12月〜2022年2月（3ヶ月）

## プロジェクト概要
エネルギー系企業向け業務改善ツールの更改対応に参画。既存機能の改修およびパフォーマンス改善を担当。

---

## 担当内容
- Python（FastAPI）によるAPI実装
- pytestを用いた単体テストの実装
- 大規模データ（100万件以上）を対象としたクエリのパフォーマンスチューニング

---

## 技術スタック
Python / FastAPI / SQLAlchemy / Pydantic / Swagger / pytest
