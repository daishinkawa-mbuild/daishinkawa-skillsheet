# 工事申請図面作成アプリケーション開発

## 期間
2024年7月〜2025年11月（1年5ヶ月）

## プロジェクト概要
公共工事申請書用の図面を作成するWebアプリケーションの開発に従事。ブラウザ上で図面を編集可能なCADライクなエディタを提供するシステムで、要件定義から設計・実装・運用まで一貫して担当。

プロジェクトにおいては、仕様整理・設計・実装まで主体的に推進し、アプリケーション全体の開発をリード。

---

## 役割・担当領域
- 要件定義から設計・実装まで一貫して担当し、開発を主導
- フロントエンド・バックエンド・インフラを横断したフルスタック開発
- 図面エディタの設計および実装を担当

---

## 実績・取り組み

### ① CADライクな図面エディタの設計・実装
- SVGをベースとしたブラウザ上の図面編集機能を設計・実装
- interact.js を用いたドラッグ・リサイズ・配置操作の実現
- 図形の配置・編集・座標管理など、CAD的な操作性を考慮したUI/ロジックを構築

---

### ② フロントエンド・バックエンドの一貫設計
- Next.js（React / TypeScript）によるフロントエンド実装
- PythonによるバックエンドAPI設計・実装
- フロントとバックエンドのデータ構造・責務分離を意識した設計

---

### ③ インフラ構築・運用
- AWS（ECS / Lambda）およびTerraformを用いたインフラ構築
- アプリケーションとインフラを含めた全体設計を担当

---

### ④ 要件整理・仕様策定の推進
- 顧客要望をもとに機能要件を整理し、システム仕様へ落とし込み
- 実装可能性を踏まえた仕様調整を行い、開発を推進

---

## 技術スタック
- フロントエンド: React / Next.js / TypeScript / interact.js / SVG
- バックエンド: Next.js / Python
- データベース: Prisma
- インフラ: AWS（ECS / Aurora / S3 / Lambda）/ Terraform

---

## 担当工程
要件定義、基本設計、詳細設計、実装、単体テスト、結合テスト、運用・保守
