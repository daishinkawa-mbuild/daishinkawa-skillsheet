# 自社クラウドを利用したNAS製品の新規開発・運用開発・保守

## 期間
2018年10月〜2020年2月（1年5ヶ月）

## プロジェクト概要
自社プロダクトであるクラウド連携型NAS製品の新規開発に従事。オンプレミス環境を基盤としたクラウドサービスとして、インフラ基盤からアプリケーションまで一貫して設計・開発を担当。

開発リーダーとして、設計・実装・テスト・リリース・運用・保守までプロダクト全体を主導。

---

## 役割・担当領域
- 開発リーダーとしてプロダクト開発を推進
- インフラ基盤（オンプレクラウド）からアプリケーションまでの設計・実装
- リリース後の運用および顧客サポート対応

---

## 実績・取り組み

### ① フルレイヤーでのプロダクト開発
- クラウド基盤、OS、バックエンド、フロントエンドまで全レイヤーを設計・実装
- OpenStackを用いたオンプレミスクラウド基盤の構築
- Docker / Kubernetes を活用したコンテナ基盤の設計・運用

---

### ② NAS製品の機能設計・実装
- Samba4 / Active Directory を用いたファイル共有機能の構築
- OpenVPN によるセキュアな接続基盤の実装
- ストレージおよび認証基盤を含むシステム全体の設計

---

### ③ 運用・監視基盤の整備
- Prometheus / Grafana を用いた監視・可視化基盤の構築
- リリース後の運用および障害対応を担当し、サービスの安定稼働を支援

---

### ④ 顧客対応およびサポート
- リリース後は顧客サポート窓口として対応
- 問い合わせ対応および問題解決を通じて、プロダクト改善に反映

---

## 技術スタック
- インフラ: OpenStack / Docker / Ansible
- ミドルウェア: Nginx / OpenVPN / Samba4 / ActiveDirectory / SWIFT
- バックエンド: Django REST Framework
- フロントエンド: Vue.js
- 監視: Prometheus / Grafana

---

## 担当工程
要件定義、基本設計、詳細設計、実装、単体テスト、結合テスト、運用・保守
