# 子供向けGPS追跡サービスの開発

## 期間
2021年7月〜2023年2月（1年8ヶ月）

## プロジェクト概要
子供向けGPS追跡サービスの新バージョン開発に従事。リアルタイムで位置情報を処理する大規模バックエンドシステムにおいて、Go言語およびAWS（Lambda / Kinesis / IoT Core）を用いた設計・実装・運用を担当。

毎分数万件規模のリクエストを処理するシステムにおいて、パフォーマンス改善および障害対応を中心に担当。

---

## 役割・担当領域
- バックエンドの設計・実装（Go / serverless構成）
- AWSを用いたリアルタイムデータ処理基盤の開発
- パフォーマンスチューニングおよび障害分析・改善対応

---

## 実績・取り組み

### ① 高負荷環境における障害分析と改善
- サービスリリース初期に、AWS Lambdaのクォータ制限に起因する障害が発生
- ログ分析およびシステム挙動の調査を行い、ボトルネックを特定
- リクエスト分散や処理設計の見直しにより、安定稼働を実現

---

### ② 性能検証の仕組み構築
- 実運用に近い負荷を再現するため、自前の性能検証ロジックを構築
- 負荷テストによりボトルネックを段階的に特定し、継続的に改善
- ボトルネック解消後に別箇所で発生する問題についても順次対応し、システム全体のスループットを向上

---

### ③ サーバレス構成におけるパフォーマンスチューニング
- Lambda / Kinesis / IoT Core を組み合わせたイベント駆動型アーキテクチャにおいて、処理効率の最適化を実施
- 高トラフィック環境における安定運用に貢献

---

## 技術スタック
- バックエンド: Go / serverless Framework
- インフラ: AWS（Lambda / RDS / Kinesis / IoT Core）

---

## 担当工程
基本設計、詳細設計、実装、結合テスト、運用・保守
