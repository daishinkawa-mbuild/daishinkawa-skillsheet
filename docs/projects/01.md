# 空間自動設計サービスのWeb開発

## 期間
2020年9月〜現在（5年5ヶ月）

## プロジェクト概要
公共トイレのレイアウトを自動生成するWebサービスの開発に長期参画。フロントエンド・バックエンド・インフラ（GCP）を横断し、設計から実装、運用まで一貫して担当。

プロダクトの中核メンバーとして、機能開発に加え、品質改善、設計改善、フルリニューアルなど複数フェーズにおいて開発を主導。

---

## 役割・ポジション
- フルスタックエンジニア（フロント / バックエンド / インフラ）
- 開発中核メンバーとして設計・実装を主導
- 顧客との仕様調整を含め、設計・実装を一貫して担当
- コードレビュー、設計改善の推進

---

## 主な取り組み

### ① 通常開発・基盤構築
- フロントエンド（Vue2 & Nuxt / React & Next）およびバックエンド（Django REST Framework）の設計・実装
- GCP（Cloud Run / Cloud SQL / Cloud Storage / Cloud Functions 等）を用いたインフラ設計・構築
- 自動設計ロジックの設計・実装

---

### ② 品質改善・リリースプロセスの再設計
- リリース事故が多発していた状況を受け、Webアプリケーションのリリース手順を見直し
- ロールバック手順の整備、リリースフローの整理を実施
- 結果として、以降の大規模リリースにおいて重大なリリース事故を抑制

---

### ③ バックエンド設計の改善（リファクタリング）
- Pythonによる既存ロジックを引き継ぎ、責務分離を意識した構造へ大幅リファクタリング
- 変更容易性・保守性を考慮した設計へ改善
- ドメインロジックとインフラ層の分離を進め、機能追加のしやすい構成へ再設計

---

### ④ フルリニューアル（デザイン〜実装まで一貫対応）
- サービスのフルリニューアルにおいて、デザイン検討段階から参画
- AIを活用したプロトタイピングにより、顧客・デザイナーと共に仕様調整を反復
- フロントエンド・バックエンドの実装からリリースまで一貫して担当

---

### ⑤ 3D/CAD機能の実装（Three.js）
- Three.js を用いたCADライクなインタラクティブ機能を実装
- 3D空間上での以下ロジックを設計・実装
  - ジオメトリ計算
  - レイキャスト
  - AABB判定
  - スナップ（吸着）処理
  - 衝突判定
- ReactベースのUIと連携し、ブラウザ上で操作可能な設計ツールを構築

---

## 技術スタック
- フロントエンド: Vue.js / Nuxt.js / React / Next.js / TypeScript / three.js
- バックエンド: Python / Django REST Framework / pytest
- インフラ: GCP（Cloud Run / Cloud Storage / Cloud SQL / Cloud Functions / Cloud Scheduler）
- CI/CD: CircleCI / Bitbucket Pipeline / Cloud Build
- その他: Docker
