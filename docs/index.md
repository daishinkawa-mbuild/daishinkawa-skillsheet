## サマリー

### キーワード
- フロントエンド / バックエンド / インフラを横断したフルスタック開発
- CADライクなアプリケーション（SVG / Three.js）など複雑なUI・ロジックの設計実装経験

- 要件整理・設計・実装を一貫して推進できる開発力
- CI/CD、E2Eテスト、リリース改善など品質向上の実績

### 経歴
エンジニアとして約13年、開発者として約10年の経験があり、Webアプリケーション開発において要件定義、設計、開発、テスト、運用まで一貫して担当してきました。単独開発から小規模チームの開発リードまで経験しています。  
Python / Django REST Framework、TypeScript / React / Next.js を主軸に、AWS・GCPを用いたクラウド環境での設計実装を担当しています。  
現在は法人を設立し、自社プロダクト開発および受託開発において設計・実装・開発推進を担当しています。

### 技術領域
業務Webシステムに加え、CADライクなUIや図面編集、3D処理などの複雑なドメインに対応可能です。  
SVGによる図面エディタや、three.jsによる3Dジオメトリ処理（レイキャスト、AABB判定、スナップ、衝突判定）など、UIとロジックを統合した実装経験があります。  
また、リニューアル案件や画面設計においては、操作性や導線を意識したUI/UX改善、デザインを踏まえたフロントエンド実装、プロトタイプを用いた仕様調整も経験しています。

### 開発スタイル
設計・実装・運用を一貫して捉え、保守性・変更容易性を重視した開発を行っています。責務分離や可読性を意識し、チームでの開発効率と品質の両立を図っています。  
AIエージェントや生成AIを活用した開発にも積極的に取り組んでおり、エージェントコーディングのみならず、設計検討、ドキュメント作成、プロトタイピングなどフルに活用しています。
