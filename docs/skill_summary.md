## 経歴
9年7ヶ月にわたり、Webアプリケーション開発においてフロントエンド、バックエンド、インフラまで一貫して担当してきました。要件定義・設計・実装・テスト・運用保守まで幅広く対応可能で、単独開発から小規模チームの開発リードまで経験しています。

Python / Django REST Framework、TypeScript / React / Next.js を主軸に、AWS・GCPを用いたクラウド環境での設計実装を担当してきました。顧客との仕様調整や不確実性の高いプロジェクトにおいても、技術的判断をもとに開発を推進できる点を強みとしています。

---

## 技術力
業務Webシステムに加え、CADライクなUIや図面編集、3D処理など、複雑なドメインを扱うシステムの設計実装を経験しています。

SVGによる図面エディタの構築や、three.jsを用いた3Dジオメトリ計算（レイキャスト、AABB判定、スナップ、衝突判定）など、インタラクティブなUIとロジックを統合した実装を担当しました。フロントエンドにとどまらず、バックエンドとのデータ構造設計やインターフェース設計まで一貫して対応しています。

また、CI/CDやE2Eテストの導入、リリースフローの改善、パフォーマンスチューニングなど、品質と運用を含めた開発基盤の構築経験があります。近年は責務分離や変更容易性を重視した設計（DDD的アプローチ）を実践しています。

---

## 開発スタイル
設計・実装・レビュー・運用までを一貫して捉え、「保守性が高く継続的に改善できるシステム」を重視しています。責務分離、可読性、変更容易性を意識し、チームでの開発効率と品質の両立を図っています。

また、AIエージェントや生成AIを開発プロセスに積極的に取り入れており、設計検討、プロトタイピング、リファクタリング、テスト生成などに活用しています。AIを活用しつつも設計判断や品質担保は自身で行い、生産性と品質のバランスを維持しています。
